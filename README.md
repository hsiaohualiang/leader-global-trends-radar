# 領導者全球趨勢雷達

這是一個給 Joyce 與同事使用的靜態網站，用來追蹤全球領導者必看的經營趨勢，並把國際財經、AI、供應鏈、治理、人才與宏觀風險訊號轉成高階主管可以討論的管理觀點。

## 使用方式

- 公開網站會由 GitHub Pages 發布。
- 網站原始檔放在 `site/`。
- 主要資料檔是 `site/data.js`。
- 每週更新後，可重新 commit 並 push，GitHub Pages 會自動重新部署。

## 內容重點

- 每週 5-7 則全球領導者必讀訊號。
- Joyce 的本週觀點與討論題。
- 25 個研究分類與趨勢卡。
- 可轉化為高階主管學習產品的 6 個議題。
- 每張趨勢卡包含來源連結、管理意義、台灣企業影響與行動建議。
